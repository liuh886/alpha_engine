# AlphaEngine 产品愿景

## 核心目标

**发掘有效的交易策略，获取 Alpha 超额收益，目标超过 QQQ（美股）/ 沪深300（A股）。**

一切功能、UI、流程都必须服务于这个目标。

## 产品原则

1. **数据驱动** - 每一个决策都必须有回测数据支撑，不接受直觉
2. **可验证** - 策略表现必须可追溯、可复现、可比较
3. **风险优先** - 控制回撤比追求收益更重要
4. **简洁直接** - UI 只展示对决策有用的信息，不做装饰

## 用户角色

**量化研究员 / 个人投资者** - 一个人完成从数据到策略到部署的全流程。

## 核心工作流

```
数据准备 → 特征工程 → 模型训练 → 回测验证 → 比较选择 → 部署竞技场 → 每日监控
```

## 页面功能与目标对齐

### 1. 首页 - 回测仪表盘 `/`

**目标：** 一眼看到"我的策略跑得怎么样"

| 区域 | 展示内容 | 对齐目标 |
|------|----------|----------|
| KPI 卡片 | 总收益、夏普比率、最大回撤、年化波动率 | vs QQQ/沪深300 的超额 |
| 权益曲线 | 策略净值 vs 基准（QQQ/CSI300） | 是否跑赢基准 |
| 持仓表 | 当前持仓、权重、行业分布 | 策略是否合理分散 |
| 归因分析 | 哪些股票贡献了 Alpha，哪些拖累了 | 理解收益来源 |

### 2. 模型管理 `/models`

**目标：** 管理和比较不同策略版本

| 功能 | 对齐目标 |
|------|----------|
| 模型列表 | 快速识别哪个版本表现最好 |
| 指标对比（年化收益、夏普、回撤） | 选择最优模型 |
| 参数展示 | 理解什么配置产生什么结果 |
| Promote/删除 | 淘汰差的，推广好的 |

### 3. 模型比较 `/compare`

**目标：** 在多个策略之间做选择

| 功能 | 对齐目标 |
|------|----------|
| 多模型指标表格 | 横向对比关键指标 |
| 归一化权益曲线 | 可视化谁跑得更好 |
| 持仓对比 | 理解策略差异 |

### 4. 竞技场 `/arena`

**目标：** 让策略在真实市场条件下竞争

| 功能 | 对齐目标 |
|------|----------|
| 排行榜 | 谁的 Alpha 最高 |
| 每日盈亏 | 实时跟踪表现 |
| 结算功能 | 定期评估淘汰 |

### 5. 数据管理 `/data`

**目标：** 确保数据质量是策略可靠性的基础

| 功能 | 对齐目标 |
|------|----------|
| 数据新鲜度 | 数据过期 = 策略不可信 |
| 质量报告 | 缺失/异常数据会影响回测 |

### 6. 报告 `/reports`

**目标：** 保存和回顾历史回测结果

| 功能 | 对齐目标 |
|------|----------|
| 回测报告列表 | 追踪策略演化历史 |

### 7. 股票终端 `/terminal`

**目标：** 深入研究个股，理解策略信号

| 功能 | 对齐目标 |
|------|----------|
| K线图 | 个股走势 |
| 模型置信度 | 策略对这只股票的看法 |
| 风险护栏 | 是否触发风控 |

### 8. 策略配置 `/strategy`

**目标：** 调整策略参数

| 功能 | 对齐目标 |
|------|----------|
| YAML 编辑器 | 修改持仓数、换仓频率、止损规则 |

### 9. 控制台 `/console`

**目标：** 执行系统操作

| 功能 | 对齐目标 |
|------|----------|
| 训练/回测触发 | 启动新的策略验证 |

| 任务监控 | 确认操作完成 |

## 关键指标（Dashboard 必须展示）

### 绝对指标
- **年化收益率** - 策略赚了多少
- **夏普比率** - 风险调整后收益（>1.0 为好，>2.0 为优秀）
- **最大回撤** - 最坏情况损失（<20% 为可接受）

### 相对指标（核心！）
- **超额收益（Alpha）** - 策略收益 - 基准收益
- **信息比率** - Alpha / 跟踪误差
- **超额最大回撤** - 策略回撤 - 基准回撤

### 风险指标
- **年化波动率** - 收益的稳定性
- **换手率** - 交易成本的代理指标
- **持仓集中度** - 前10大持仓权重占比

## UI 设计准则

1. **Alpha 第一** - 任何页面最显眼的数字必须是超额收益，不是绝对收益
2. **基准对比** - 每张图表都必须有 QQQ/沪深300 基准线
3. **红绿分明** - 跑赢基准 = 绿色，跑输 = 红色
4. **一屏信息** - 首页不需要滚动就能看到所有关键指标
5. **操作可达** - 从看到问题到采取行动（训练/回测/更新数据）不超过 2 次点击

## 不做什么

- 不做实时交易（只做回测和模拟竞技场）
- 不做多用户/权限系统（单人使用）
- 不做花哨的动画和3D图表
- 不做新闻/舆情分析（纯量化因子模型）
- 不做期权/衍生品（只做股票多头）
